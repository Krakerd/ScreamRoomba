# ScreamRoomba
Este Programa esta escrito en cpp para su uso con plataform.io (https://platformio.org), la función del mismo es a partir de dos finales de carrera detectar las colisiones de una roomba o similar.

Esquema de pines (finales de carrera):
  Final de Carrera1: pin 12
  Final de Carrera2: pin 13

Esquema de pines (DFplayer mini):
  RX en arduino: 10
  TX en arduino: 11
  Conexiones al chip: se invierte los pines (la líe con los nombres de variables y soy muy vago como para cambiarlo)
